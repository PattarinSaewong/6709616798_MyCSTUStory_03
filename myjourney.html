My Journey

Year 1

Fundamental Strengthening

  • Programming foundations: Learn languages such as C, HTML, and CSS from online platforms like W3Schools.
  • Supporting skills: Focus on English and mathematics by actively participating in class and consistently reviewing content.

Year 2

Proficient in Programming

  • Practice using previously learned concepts and think outside the box.
  • Apply what is taught in class by trying to write code independently.
  • Study online resources, such as websites that aggregate programming languages and YouTubers who provide programming tutorials.
  • Improve attention to detail and continuously refine your coding style.

Year 3

Specialized field of study

  • Up Hard Skills: Take online courses related to computer science skills.
  • Develop Soft Skills: These include creativity, decision-making, and problem-solving.

Year 4

Graduate with Honors

  • Practice using previously learned concepts and think outside the box.
  • Work on a project related to smart homes, where everything in the house can be connected and controlled through a single app to make daily life more convenient.
